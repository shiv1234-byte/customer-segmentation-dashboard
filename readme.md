# Customer Segmentation & Retention Dashboard

This project analyzes customer and purchase data using RFM analysis and machine learning clustering (Python), visualizes insights in Excel and Power BI, and delivers actionable retention strategies.

## Contents
- Sample data
- Python scripts for RFM & clustering
- Excel and Power BI dashboards
- Final report

## Tools Used
- Python (pandas, scikit-learn)
- Microsoft Excel
- Power BI (Tableau recommended for additional viz experience)
- GitHub

## How to Run
1. Clone this repo
2. Load `customer_data.csv` & `purchase_data.csv` in Excel/Python
3. Run `python/rfm_analysis.py` for segmentation
4. Open `dashboard.pbix` in Power BI
5. Read final report for recommendations
